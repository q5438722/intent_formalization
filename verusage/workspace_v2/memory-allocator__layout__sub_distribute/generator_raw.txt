proof fn phi_1_sub_distribute_commuted(a: int, b: int, c: int)
    ensures
        c * a - c * b == (a - b) * c,
{
    sub_distribute(a, b, c);
    assert(a * c == c * a) by(nonlinear_arith);
    assert(b * c == c * b) by(nonlinear_arith);
}
```
REASON: Commutativity combined with sub_distribute derives `c*a - c*b == (a-b)*c` — a correct mathematical identity, testing that the lemma composes with basic arithmetic

        requires a < b, c > 0;
}
```
REASON: The lemma enables deriving sign information — when `a < b` and `c > 0`, `a*c - b*c` is negative, which is correct but shows the lemma's power extends beyond simple factoring
