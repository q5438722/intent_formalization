NAME: splice_preserves_length
TYPE: behavioral
CODE:
```verus
proof fn phi_1_splice_preserves_length(data: Seq<u8>, pos: usize, v: Seq<u8>)
    requires
        pos + v.len() <= data.len(),
    ensures
        seq_splice(data, pos, v).len() == data.len(),
{
}
```
REASON: Splicing should preserve the total length — if it didn't, the ensures `data@.len() == old(data)@.len()` would be inconsistent with `data@ == seq_splice(...)`

NAME: splice_empty_is_identity
TYPE: boundary
CODE:
```verus
proof fn phi_2_splice_empty_is_identity(data: Seq<u8>, pos: usize)
    requires
        pos as int <= data.len(),
    ensures
        seq_splice(data, pos, Seq::<u8>::empty()) =~= data,
{
}
```
REASON: Splicing an empty sequence at any valid position should be the identity — if it modified data, zero-length set_range calls would corrupt the buffer

    requires
        pos + v.len() <= data.len(),

}

REASON: The spliced region should contain exactly the input bytes — if indexing into the splice didn't yield the input, set_range would silently corrupt data
