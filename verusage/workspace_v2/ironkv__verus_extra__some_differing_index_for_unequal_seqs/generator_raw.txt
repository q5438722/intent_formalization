NAME: swap_produces_different_seq
TYPE: behavioral
CODE:
```verus
proof fn phi_3_swap_produces_different_seq(s: Seq<int>, i: int, j: int)
    requires
        0 <= i < s.len(),
        0 <= j < s.len(),
        i != j,
        s[i] != s[j],
    ensures
        s != s.update(i, s[j]).update(j, s[i]) || s[i] == s[j],
{
    let s2 = s.update(i, s[j]).update(j, s[i]);
    if s == s2 {
        assert(s[i] == s2[i]);
    }
}
```
REASON: Swapping two distinct elements should produce a different sequence — if it didn't, the function's ability to find the differing index would be undermined
