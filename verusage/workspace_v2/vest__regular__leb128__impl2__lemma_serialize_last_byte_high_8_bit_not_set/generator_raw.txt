NAME: serialize_127_is_single_byte
TYPE: behavioral
CODE:
```verus
proof fn phi_5_serialize_127_is_single_byte()
    ensures
        UnsignedLEB128.spec_serialize(127u64) == seq![127u8],
{
    assert(127u64 >> 7u64 == 0u64) by (bit_vector);
    assert(take_low_7_bits!(127u64) == 127u8) by (bit_vector);
}
```
REASON: 127 (0x7F) is the largest single-byte LEB128 value — if it required two bytes, the encoding boundary between single and multi-byte would be wrong
