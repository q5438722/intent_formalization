```verus
proof fn phi_2_insert_propagates_to_interp_aux(d1: Directory, d2: Directory, idx: nat, vaddr: nat, pte: PTE)
    requires
        d1.inv(),
        d2.inv(),
        idx < d1.entries.len(),
        d1.arch == d2.arch,
        d1.layer == d2.layer,
        d1.base_vaddr == d2.base_vaddr,
        d2.entries[idx as int].interp(d1.entry_base(idx))
            == d1.entries[idx as int].interp(d1.entry_base(idx)).insert(vaddr, pte),
        forall|j: nat| j < d1.entries.len() && j != idx
            ==> (#[trigger] d1.entries[j as int]).interp(d1.entry_base(j))
                        == d2.entries[j as int].interp(d1.entry_base(j)),
    ensures
        d2.interp_aux(0) === d1.interp_aux(0).insert(vaddr, pte),
{
    d1.lemma_entries_interp_insert_implies_interp_aux_insert(d2, idx, vaddr, pte, 0);
}
```
REASON: Single-entry insertion propagates to interp_aux — the verified proof depends on the external_body disjointness lemma for the `idx > i` case
