NAME: upper_vaddr_not_canonical
TYPE: behavioral
CODE:
```verus
proof fn phi_3_upper_vaddr_not_canonical()
    ensures
        x86_arch_spec.upper_vaddr(0, 0) > (1usize << 48) as nat,
{
    x86_arch_spec_upper_bound();
}
```
REASON: `upper_vaddr` computes to 256TB (512^2 * 1G = 2^48) which equals the 48-bit canonical address boundary — addresses at or above this are non-canonical on x86-64 without LA57
===PHI_END===
