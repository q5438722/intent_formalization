```verus
proof fn phi_5_empty_vec_serialize_not_empty()
    ensures
        ({
            let v: Vec<u8> = Vec::new();
            v.is_marshalable() && v.ghost_serialize().len() > 0
        }),
{
    let v: Vec<u8> = Vec::new();
    lemma_auto_spec_u64_to_from_le_bytes();
    assert(v@.len() == 0);
    assert(v.is_marshalable());
    assert((0usize).ghost_serialize().len() == 8);
}
```
REASON: An empty Vec<u8> serializes to 8 bytes (the length prefix) rather than 0 — the serialization format has non-trivial overhead that makes `ghost_serialize().len() == 0` impossible for any Vec, potentially surprising consumers expecting empty containers to have empty serializations
