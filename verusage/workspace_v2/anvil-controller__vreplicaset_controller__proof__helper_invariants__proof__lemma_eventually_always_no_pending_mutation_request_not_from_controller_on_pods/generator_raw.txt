```verus
proof fn phi_2_in_flight_unique_ids_implies_singleton(s: ClusterState, msg1: Message, msg2: Message)
    requires
        Cluster::every_in_flight_msg_has_unique_id()(s),
        s.in_flight().contains(msg1),
        s.in_flight().contains(msg2),
        msg1.rpc_id == msg2.rpc_id,
    ensures
        msg1 == msg2,
        s.in_flight().count(msg1) == 1,
{
}
```
REASON: unique IDs imply msg1==msg2 but NOT that the multiset count is 1 — a Multiset can contain duplicate entries of the same message, and the spec only constrains distinct messages
