NAME: different_vrs_disjoint_matching_pods
TYPE: logical
CODE:
```verus
proof fn phi_4_different_vrs_disjoint_matching_pods(vrs1: VReplicaSetView, vrs2: VReplicaSetView, s: StoredState)
    requires
        vrs1.metadata.uid != vrs2.metadata.uid,
        vrs1.metadata.name != vrs2.metadata.name,
    ensures
        matching_pods(vrs1, s).disjoint(matching_pods(vrs2, s)),
{
}
```
REASON: a pod's owner_references is a Seq that can contain multiple entries — a pod could have owner refs for both controllers, so different VRS do NOT imply disjoint matching pod sets
===PHI_END===
