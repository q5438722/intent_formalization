NAME: push_preserves_prefix_state
TYPE: behavioral
CODE:
```verus
proof fn phi_1_push_preserves_prefix_state<DT: Dispatch>(
    a: Seq<DT::WriteOperation>,
    x: DT::WriteOperation,
    i: LogIdx,
)
    requires
        i <= a.len(),
    ensures
        compute_nrstate_at_version::<DT>(a, i) == compute_nrstate_at_version::<DT>(a.push(x), i),
{
    state_at_version_preserves::<DT>(a, a.push(x), x, i);
}
```
REASON: Appending an operation preserves state at all prior versions — the new operation is invisible to prefix computations

    }),
