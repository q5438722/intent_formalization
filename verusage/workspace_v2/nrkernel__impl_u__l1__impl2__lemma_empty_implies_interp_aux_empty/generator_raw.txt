NAME: interp_aux_right_overrides_left
TYPE: behavioral
CODE:
```verus
proof fn phi_3_interp_aux_right_overrides_left(d: Directory)
    requires
        d.inv(),
        d.entries.len() >= 2,
        d.entries[0] is Page,
        d.entries[1] is Page,
    ensures
        // interp_aux uses union_prefer_right with i=0 entry on top
        // entry 0's interp overrides entry 1's if keys collide
        d.interp_aux(0) == d.interp_aux(1).union_prefer_right(d.interp_of_entry(0)),
{
}
```
REASON: `interp_aux` builds the map with earlier entries overriding later ones via `union_prefer_right` — key collisions silently shadow entries without the spec proving disjointness

```verus
        d.entries[i as int] is Directory,
    ensures
        // subdirectory base = parent base + i * entry_size, but parent base itself may be unaligned
        d.entries[i as int]->Directory_0.base_vaddr == d.base_vaddr + i * d.entry_size(),
{
}
```
REASON: Subdirectory `base_vaddr` is computed from parent's `base_vaddr` which has no alignment constraint — misalignment propagates through the tree
