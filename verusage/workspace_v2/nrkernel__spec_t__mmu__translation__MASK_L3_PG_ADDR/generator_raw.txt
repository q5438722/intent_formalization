NAME: mask_l3_pg_addr_low_12_bits_zero
TYPE: boundary
CODE:
```verus
proof fn phi_1_mask_l3_pg_addr_low_12_bits_zero()
    ensures
        MASK_L3_PG_ADDR_SPEC & 0xFFF == 0,
{
    axiom_max_phyaddr_width_facts();
    assert(forall|low: usize, high: usize| low <= high && high < 64 ==> bitmask_inc!(low, high) & (!0usize >> (64 - low)) == 0) by (bit_vector);
}
```
REASON: Bits 0-11 should be clear for 4KB page address extraction — if any low bits leak, physical addresses would include sub-page offsets
