```verus
proof fn phi_3_delegate_for_key_range_contradictory_dsts(dm: DelegationMap<AbstractKey>, k: AbstractKey, dst1: ID, dst2: ID)
    requires
        dm.valid(),
        dst1@ != dst2@,
    ensures
        AbstractDelegationMap::delegate_for_key_range_is_host(
            AbstractDelegationMap(dm@),
            KeyRange { lo: KeyIterator::new_spec(k), hi: KeyIterator::new_spec(k) },
            dst1@,
        ),
        AbstractDelegationMap::delegate_for_key_range_is_host(
            AbstractDelegationMap(dm@),
            KeyRange { lo: KeyIterator::new_spec(k), hi: KeyIterator::new_spec(k) },
            dst2@,
        ),
{
}
```
REASON: A degenerate range [k, k) is simultaneously "delegated" to two different hosts with different abstract endpoints, because `between(new_spec(k), new_spec(k), new_spec(k))` requires `k < k` which is false — callers cannot distinguish "all keys agree" from "empty range"
