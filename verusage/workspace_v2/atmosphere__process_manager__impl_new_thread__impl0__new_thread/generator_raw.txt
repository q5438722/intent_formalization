```verus
proof fn phi_3_thread_running_cpu_container_mismatch(pm: ProcessManager, t_ptr: ThreadPtr, cpu_i: CpuId)
    requires
        pm.cpus_wf(),
        pm.threads_cpu_wf(),
        pm.container_perms_wf(),
        pm.container_cpu_wf(),
        pm.thread_dom().contains(t_ptr),
        pm.get_thread(t_ptr).state == ThreadState::RUNNING,
        pm.get_thread(t_ptr).running_cpu == Some(cpu_i),
        0 <= cpu_i < NUM_CPUS,
    ensures
        pm.cpu_list@[cpu_i as int].owning_container == pm.get_thread(t_ptr).owning_container,
{
}
```
REASON: A running thread's owning_container should match its CPU's owning_container — if this isn't entailed, a thread could run on a CPU belonging to a different container, breaking isolation
