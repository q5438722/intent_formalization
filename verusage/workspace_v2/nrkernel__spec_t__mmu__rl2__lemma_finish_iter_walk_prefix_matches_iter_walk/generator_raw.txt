```verus
    requires
        walk.path.len() >= 4,
        !walk.complete,
    ensures
        ({
            reveal(walk_next);
            walk_next(mem, walk).path.len() == walk.path.len() + 1
        }),
{
    reveal(walk_next);
}
```
REASON: `walk_next` on a walk with path.len() >= 4 uses `arbitrary()` for the address — the walk continues with an unspecified address, producing a 5+ length path that exceeds the 4-layer page table
