NAME: subtract_consecutive_multiples
TYPE: logical
CODE:
```verus
proof fn phi_4_subtract_consecutive_multiples(c: nat, k1: nat, k2: nat)
    requires
        c > 0,
        k1 <= k2,
    ensures
        aligned((k2 * c - k1 * c) as nat, c),
{
    assert(aligned(k1 * c, c)) by {
        assert((k1 * c) % c == 0) by (nonlinear_arith) requires c > 0;
    };
    assert(aligned(k2 * c, c)) by {
        assert((k2 * c) % c == 0) by (nonlinear_arith) requires c > 0;
    };
    assert(k1 * c <= k2 * c) by (nonlinear_arith) requires k1 <= k2, c > 0;
    subtract_mod_eq_zero(k1 * c, k2 * c, c);
}
```
REASON: Difference of any two multiples of `c` is aligned to `c` — tests closure of multiples under subtraction
===PHI_END===
