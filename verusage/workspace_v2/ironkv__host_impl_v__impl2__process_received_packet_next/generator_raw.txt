NAME: option_tag_collides_with_data
TYPE: logical
CODE:
```verus
proof fn phi_5_option_tag_collides_with_data(v: Vec<u8>)
    requires
        v.is_marshalable(),
        v@.len() > 0,
        v@[0] == 0u8,
    ensures
        ({
            let some_v: Option<Vec<u8>> = Some(v);
            some_v.ghost_serialize()[0] == 1u8
        }),
        ({
            let none_v: Option<Vec<u8>> = None;
            none_v.ghost_serialize()[0] == 0u8
        }),
{
}
```
REASON: Option serialization uses a single tag byte (0 for None, 1 for Some) with no framing — while the tag itself distinguishes None/Some, there is no length field after the tag, so a deserializer seeing `[1, 0, ...]` cannot distinguish `Some(v)` where `v` serializes starting with 0 from a corruption scenario without knowing the inner type's exact deserialized length
===PHI_END===
