NAME: container_map_4k_not_subset_allocated
TYPE: logical
CODE:
```verus
proof fn phi_1_container_map_4k_not_subset_allocated(
    pa: PageAllocator,
    c: ContainerPtr,
    p: PagePtr,
)
    requires
        pa.wf(),
        pa.container_map_4k@.dom().contains(c),
        pa.container_map_4k@[c].contains(p),
        !pa.allocated_pages_4k@.contains(p),
    ensures
        pa.container_map_4k@[c].contains(p),
{
}
```
REASON: container_map_4k tracks per-container page ownership but wf() may not require these pages to actually be in allocated_pages_4k — containers could claim unallocated pages
===PHI_END===

```verus
    pa: PageAllocator,
    p: PagePtr,
)
    requires
        pa.wf(),
        pa.mapped_pages_4k@.contains(p),
        pa.page_mappings(p) =~= Set::empty(),
    ensures
        pa.mapped_pages_4k@.contains(p),
{
}
```
REASON: A page can be in Mapped4k state yet have an empty mappings set — the spec may not require mapped pages to have at least one mapping recorded

NAME: merged_2m_parent_could_be_unavailable
TYPE: boundary
CODE:
```verus
proof fn phi_5_merged_2m_parent_could_be_unavailable(
    pa: PageAllocator,
    i: usize,
)
    requires
        pa.wf(),
        0 <= i < NUM_PAGES,
        pa.page_array@[i as int].state == PageState::Merged2m,
        pa.page_array@[spec_page_index_truncate_2m(i) as int].state == PageState::Unavailable2m,
    ensures
        pa.page_array@[i as int].state == PageState::Merged2m,
{
}
```
REASON: merged_pages_wf allows a Merged2m page's parent to be Unavailable2m — a merged sub-page of an unavailable 2M page is a contradictory state that should not be allowed
