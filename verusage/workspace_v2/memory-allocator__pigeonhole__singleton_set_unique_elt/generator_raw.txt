proof fn phi_4_remove_from_singleton_empty(s: Set<nat>, a: nat)
    requires
        s.finite(),
        s.len() == 1,
        s.contains(a),
    ensures
        s.remove(a) =~= Set::<nat>::empty(),
{
    let r = s.remove(a);
    assert(r.len() == 0);
    assert forall |x: nat| !r.contains(x) by {}
}
```
REASON: Removing the sole element from a singleton yields empty — tests the boundary interaction between `remove` and `len`

    requires
        s.finite(),
        s.len() == 1,
        s.contains(a),
        c != a,
    ensures

        singleton_set_unique_elt(s, a, c);
    }

```
REASON: A singleton set cannot contain a distinct element — correct contrapositive of uniqueness, but confirms the lemma's full power
