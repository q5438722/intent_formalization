```verus
proof fn phi_4_always_propagate_unbounded(ex: Execution<int>, p: TempPred<int>)
    requires
        always(p).satisfied_by(ex),
    ensures
        forall |i: nat, j: nat| p.satisfied_by(#[trigger] ex.suffix(i).suffix(j)),
{
    assert forall |i: nat, j: nat| p.satisfied_by(#[trigger] ex.suffix(i).suffix(j)) by {
        always_propagate_forwards::<int>(ex, p, i);
        always_unfold::<int>(ex.suffix(i), p);
    };
}
```
REASON: the two external_body axioms together yield a universally quantified double-suffix property — this unbounded composition amplifies trust surface since suffix nesting never reduces to a single offset without an explicit composition lemma
