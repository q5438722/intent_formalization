NAME: distinct_entries_disjoint_ranges
TYPE: behavioral
CODE:
```verus
proof fn phi_3_distinct_entries_disjoint_ranges(d: Directory, i: nat, j: nat, va: nat)
    requires
        d.inv(),
        i < j,
        j < d.entries.len(),
        d.interp_of_entry(i).contains_key(va),
    ensures
        !d.interp_of_entry(j).contains_key(va),
{
    d.lemma_interp_of_entry_key_between(i, va);
    if d.interp_of_entry(j).contains_key(va) {
        d.lemma_interp_of_entry_key_between(j, va);
    }
}
```
REASON: Two applications of the external_body key-between axiom derive entry disjointness — if the bounds axiom were wrong, overlapping entries could silently shadow each other

```verus
proof fn phi_4_remove_propagates_to_interp_aux(d1: Directory, d2: Directory, idx: nat, vaddr: nat)
    requires
        d1.inv(),
        d2.inv(),
        idx < d1.entries.len(),
        d1.arch == d2.arch,
        d1.layer == d2.layer,
        d1.base_vaddr == d2.base_vaddr,
        d1.entries[idx as int].interp(d1.entry_base(idx)).contains_key(vaddr),
        d2.entries[idx as int].interp(d1.entry_base(idx)) == d1.entries[idx as int].interp(d1.entry_base(idx)).remove(vaddr),
        forall|j: nat| j < d1.entries.len() && j != idx
            ==> (#[trigger] d1.entries[j as int]).interp(d1.entry_base(j))
                        == d2.entries[j as int].interp(d1.entry_base(j)),
    ensures
        d2.interp_aux(0) === d1.interp_aux(0).remove(vaddr),
{
    d1.lemma_entries_interp_remove_implies_interp_aux_remove(d2, idx, vaddr, 0);
}
```
REASON: Remove propagation depends on both external_body axioms — the verified proof uses key-between and contains-implies-entry-contains as unverified foundations
