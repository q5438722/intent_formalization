```verus
proof fn phi_4_parse_high_bit_result_greater_than_one(s: Seq<u8>)
    requires
        s.len() > 0,
        s[0] as u8 >= 0x80,
        UnsignedLEB128.spec_parse(s).is_some(),
    ensures
        UnsignedLEB128.spec_parse(s).unwrap().1 > 1,
{
    UnsignedLEB128.lemma_spec_parse_non_zero(s);
}
```
REASON: When the first byte has high bit set and parse succeeds, the result must be > 1 — this is proven by a verified lemma, but the bound `> 1` rather than `>= 128` seems surprisingly weak; if the lower bound were actually higher, the lemma would be hiding a tighter invariant
