```verus
proof fn phi_1_lemma_nat_range_external_body(lo: nat, hi: nat)
    requires
        lo <= hi,
    ensures
        set_nat_range(lo, hi).finite(),
        set_nat_range(lo, hi).len() == hi - lo,
{
    lemma_nat_range(lo, hi);
}
```
REASON: `lemma_nat_range` is `external_body` — finiteness and length of the nat range set are trusted without proof; if the axiom were wrong, `nat_set_size` would inherit the unsoundness

```verus
proof fn phi_3_nat_set_size_uniqueness(s1: Set<nat>, s2: Set<nat>, bound: nat)
    requires
        forall |i: nat| (0 <= i < bound <==> s1.contains(i)),
        forall |i: nat| (0 <= i < bound <==> s2.contains(i)),
    ensures
        s1 =~= s2,
{
    nat_set_size(s1, bound);
    nat_set_size(s2, bound);
}
```
REASON: Any two sets satisfying the same `[0, bound)` membership are extensionally equal — follows from the `=~=` assertion in `nat_set_size`, but the proof is powered by the external_body axiom
===PHI_END===
