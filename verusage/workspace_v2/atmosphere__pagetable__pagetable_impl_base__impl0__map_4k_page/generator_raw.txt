        pe.perm.present == true,
    ensures
        spec_page_entry_to_map_entry(&pe) == spec_page_entry_to_map_entry(

NAME: user_only_no_l1_constraint
TYPE: behavioral
CODE:
```verus
proof fn phi_4_user_only_no_l1_constraint(pt: PageTable, p: PageMapPtr, i: L1Index)
    requires
        pt.user_only(),
        pt.wf_l1(),
        pt.l1_tables@.dom().contains(p),
        0 <= i < 512,
        pt.l1_tables@[p].value()[i].perm.present,
    ensures
        pt.l1_tables@[p].value()[i].perm.user
        || !pt.l1_tables@[p].value()[i].perm.user,
{
}
```
REASON: user_only constrains L4/L3/L2 entries to have user bit set when present, but if L1 entries are not similarly constrained, a kernel-only 4K page could exist under an otherwise user-accessible subtree
===PHI_END===
