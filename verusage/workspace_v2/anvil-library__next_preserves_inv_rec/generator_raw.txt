```verus
proof fn phi_4_inv_preserved_for_all_indices(ex: Execution<int>, next: TempPred<int>, inv: TempPred<int>)
    requires
        inv.satisfied_by(ex),
        forall |idx: nat| next.satisfied_by(#[trigger] ex.suffix(idx)),
        forall |idx: nat| inv.satisfied_by(#[trigger] ex.suffix(idx)) && next.satisfied_by(ex.suffix(idx))
            ==> inv.satisfied_by(ex.suffix(idx + 1)),
    ensures
        forall |i: nat| inv.satisfied_by(#[trigger] ex.suffix(i)),
{
    assert forall |i: nat| inv.satisfied_by(#[trigger] ex.suffix(i)) by {
        next_preserves_inv_rec::<int>(ex, next, inv, i);
    };
}
```
REASON: next_preserves_inv_rec generalizes to a universal quantifier over all indices — if execution_equality is unsound, the entire inductive invariant preservation is compromised
