```verus
proof fn phi_1_wherever_no_writes_external_body(
    pm_region_view: PersistentMemoryRegionView,
    s: Seq<u8>,
    addr: int,
)
    requires
        pm_region_view.can_crash_as(s),
        0 <= addr < s.len(),
        pm_region_view.state[addr].outstanding_write.is_none(),
    ensures
        s[addr] == pm_region_view.committed()[addr],
{
    lemma_wherever_no_outstanding_writes_persistent_memory_view_can_only_crash_as_committed(pm_region_view);
}
```
REASON: `lemma_wherever_no_outstanding_writes_persistent_memory_view_can_only_crash_as_committed` is `external_body` — the per-address crash determinism for bytes without outstanding writes is trusted without proof; if a chunk with mixed outstanding/non-outstanding bytes could crash as flushed, the non-outstanding byte would change
