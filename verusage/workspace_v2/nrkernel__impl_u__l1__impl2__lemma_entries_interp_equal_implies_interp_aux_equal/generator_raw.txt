```verus
proof fn phi_1_interp_aux_equal_from_entries(d1: Directory, d2: Directory)
    requires
        d1.inv(),
        d2.inv(),
        d1.arch == d2.arch,
        d1.layer == d2.layer,
        d1.base_vaddr == d2.base_vaddr,
        forall|j: nat| j < d1.entries.len()
            ==> (#[trigger] d1.entries[j as int]).interp(d1.entry_base(j)) == d2.entries[j as int].interp(d1.entry_base(j)),
    ensures
        d1.interp_aux(0) === d2.interp_aux(0),
{
    d1.lemma_entries_interp_equal_implies_interp_aux_equal(d2, 0);
}
```
REASON: Two directories with same entry interpretations have the same interp — direct consequence of the verified lemma
