```verus
proof fn phi_4_monotonic_does_not_require_key_preservation(base: nat, pte1: PTE, pte2: PTE)
    requires
        pte1.frame.size > 0,
        pte2.frame.size > 0,
        overlap(MemRegion { base: base, size: pte1.frame.size }, MemRegion { base: base, size: pte2.frame.size }),
    ensures
        candidate_mapping_overlaps_existing_vmem(
            Map::empty().insert(base, pte1).insert(base, pte2),
            base,
            pte2,
        ),
{
}
```
REASON: Submap monotonicity holds even when mappings2 overwrites a key from mappings1 — the lemma's `submap_of` precondition doesn't account for value changes at existing keys
