```verus
proof fn phi_3_entails_transitive_via_apply(p: TempPred<int>, q: TempPred<int>, r: TempPred<int>)
    requires
        p.entails(q),
        q.entails(r),
    ensures
        p.entails(r),
{
    assert forall |ex: Execution<int>| #[trigger] p.satisfied_by(ex) implies r.satisfied_by(ex) by {
        entails_apply::<int>(ex, p, q);
        entails_apply::<int>(ex, q, r);
    };
}
```
REASON: derives entailment transitivity by chaining entails_apply — if the closure bridge resolves incorrectly, this enables chaining arbitrary entailment claims
