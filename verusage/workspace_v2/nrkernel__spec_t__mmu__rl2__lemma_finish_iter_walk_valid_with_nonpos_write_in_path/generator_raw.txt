```verus
proof fn phi_1_walk_next_preserves_vaddr(mem: PTMem, walk: Walk)
    requires
        !walk.complete,
        walk.path.len() < 4,
    ensures
        ({
            reveal(walk_next);
            walk_next(mem, walk).vaddr == walk.vaddr
        }),
{
    reveal(walk_next);
}
```
REASON: `walk_next` should preserve the virtual address being walked — if it changes vaddr, the walk diverges from its original translation target

proof fn phi_3_finish_iter_walk_always_complete(mem: PTMem, walk: Walk)

    ensures

    reveal(walk_next);

REASON: Starting from an empty incomplete walk, `finish_iter_walk` applies up to 4 steps — the walk should always complete since every non-Directory PDE sets complete=true and after 4 levels there are no more Directory options
