NAME: page_align_not_word_align
TARGET_FN: aligned
TYPE: logical
CODE:
```verus
proof fn phi_3_page_align_not_word_align(addr: nat)
    requires
        aligned(addr, 4096),
    ensures
        !aligned(addr, 8),
{
}
```
REASON: Page-alignment (4096) must imply 8-byte alignment since 4096 is a multiple of 8; if it doesn't, pml4 alignment invariants fail to guarantee write alignment.
