TARGET_FN: align_to_usize
TYPE: behavioral
CODE:
```verus
proof fn phi_1_align_always_zero(a: usize, b: usize)

        b > 0,
        a >= b,
    ensures

}
```
REASON: If alignment collapses every value >= b to zero, the function destroys address information and is trivially wrong.
