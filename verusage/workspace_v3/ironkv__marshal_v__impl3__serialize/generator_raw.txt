```verus
proof fn phi_1_none_serialize_empty(pre_data: Seq<u8>, post_data: Seq<u8>)
    requires
        Option::<u64>::None.is_marshalable(),
        post_data.len() >= pre_data.len(),
        post_data.subrange(0, pre_data.len() as int) == pre_data,
        post_data.subrange(pre_data.len() as int, post_data.len() as int) == Option::<u64>::None.ghost_serialize(),
    ensures
        post_data.len() == pre_data.len(),
{
}
```
REASON: Serializing None should append a tag byte, so output length must grow — being equal would mean nothing was written.
