NAME: new_contains_zero
TARGET_FN: new
TYPE: behavioral
CODE:
```verus
proof fn phi_1_new_contains_zero<const N: usize>(ret: ArraySet<N>)
    requires
        ret.wf(),
        ret@ == Set::<usize>::empty(),
        N > 0,
    ensures
        ret@.contains(0usize),
{
}
```
REASON: An empty set should not contain element 0; if provable, the spec fails to exclude membership.
