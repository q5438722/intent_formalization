TARGET_FN: aligned
TYPE: logical
CODE:
```verus
proof fn phi_3_successor_preserves_page_alignment(addr: nat)
    requires
        aligned(addr, PAGE_SIZE as nat),
    ensures
        aligned((addr + 1) as nat, PAGE_SIZE as nat),

}
```
REASON: Adding 1 to a page-aligned address must NOT remain page-aligned; if it does the spec conflates all addresses

NAME: double_size_always_entailed

        s > 0,

}
