NAME: result_equals_base
TARGET_FN: ArchExec::next_entry_base
TYPE: behavioral
CODE:
```verus
proof fn phi_1_result_equals_base(arch: ArchExec, layer: usize, base: usize)
    requires
        arch@.inv(),
        layer < arch@.layers.len(),
        base <= MAX_BASE,
        0usize <= X86_NUM_ENTRIES,
    ensures
        arch@.next_entry_base(layer as nat, base as nat, 0nat) == base as nat,
{
}
```
REASON: If next_entry_base with idx=0 equals base, entries would start at the same address as the base, meaning no address space is actually mapped by the first entry.
