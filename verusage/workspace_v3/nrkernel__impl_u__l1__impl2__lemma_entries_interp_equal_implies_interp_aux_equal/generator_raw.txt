TYPE: logical
CODE:
```verus
proof fn phi_3_entry_base_collision_distinct_indices(arch: Arch, layer: nat, base: nat, i: nat, j: nat)
    requires
        arch.inv(),
        layer < arch.layers.len(),
        i < arch.num_entries(layer),
        j < arch.num_entries(layer),
        i != j,
    ensures
        arch.entry_base(layer, base, i) == arch.entry_base(layer, base, j),
{
}
```
REASON: If two distinct entry indices produced the same base address, page table entries would alias and the MMU mapping would be ambiguous.
