NAME: insert_removes_unrelated_key
TARGET_FN: insert
TYPE: behavioral
CODE:
```verus
proof fn phi_1_insert_removes_unrelated_key(
    pre: TombstoneTable, post: TombstoneTable,
    src: AbstractEndPoint, last_seqno: u64,
    other: AbstractEndPoint,
)
    requires
        forall |k: AbstractEndPoint| #[trigger] pre.contains_key(k) ==> k.valid_physical_address(),
        src.valid_physical_address(),
        post =~= pre.insert(src, last_seqno as nat),
        pre.contains_key(other),
        other != src,
    ensures
        !post.contains_key(other),
{
}
```
REASON: Insert should preserve all pre-existing keys unrelated to src; losing them would silently drop tombstone entries
===PHI_END===
