TARGET_FN: v2l2index
TYPE: behavioral
CODE:
```verus
proof fn phi_4_l2index_always_zero_2m(va: usize)
    requires
        spec_va_2m_valid(va),
    ensures
        spec_v2l2index(va) == 0,
{
}
```
REASON: 2M alignment zeros bits 0–20 but bits 21–29 are free; if the index were always 0, the L2 table could only map one 2M region per L3 entry
