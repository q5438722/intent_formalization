NAME: inv_allows_equal_consecutive_entry_sizes
TARGET_FN: entry_size_is_next_layer_size
TYPE: logical
CODE:
```verus
proof fn phi_4_inv_allows_equal_consecutive_entry_sizes()
    ensures
        ({
            let arch = Arch { layers: seq![
                ArchLayer { entry_size: 4096, num_entries: 1 },
                ArchLayer { entry_size: 4096, num_entries: 1 },
            ] };
            arch.inv() && arch.entry_size(0) == arch.entry_size(1)
        }),
{
}
```
REASON: Consecutive layers having identical entry sizes (via num_entries=1) defeats the hierarchical structure; each deeper layer should map a strictly smaller region.
===PHI_END===
