```verus
proof fn phi_4_entry_base_collision(arch: Arch, layer: nat, base: nat, i: nat, j: nat)
    requires
        arch.inv(),
        layer < arch.layers.len(),
        i < arch.num_entries(layer),
        j < arch.num_entries(layer),
        i != j,
    ensures
        arch.entry_base(layer, base, i) == arch.entry_base(layer, base, j),
{
}
```
REASON: Distinct entry indices mapping to the same base address would collapse the entire page table structure, making separate pages indistinguishable

```verus
proof fn phi_5_aligned_implies_double_aligned(addr: nat, size: nat)

    ensures

}
```

REASON: Alignment to size does not imply alignment to 2*size; if provable it would mean every 4K-aligned address is also 8K-aligned, breaking page-size distinctions
