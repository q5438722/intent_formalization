CODE:
```verus

    requires
        arch.inv(),
        layer < arch.layers.len(),
        base > 0,
    ensures
        arch.upper_vaddr(layer, base) <= base,
{
}
```
REASON: upper_vaddr should always exceed base for a valid layer, since the mapped region has positive size; equality or less implies an empty or wrapping address space
