NAME: entry_base_collapses_indices
TARGET_FN: entry_base
TYPE: logical
CODE:
```verus
proof fn phi_4_entry_base_collapses_indices(arch: Arch, layer: nat, base: nat, i: nat, j: nat)
    requires
        arch.inv(),
        layer < arch.layers.len(),
        i < arch.num_entries(layer),
        j < arch.num_entries(layer),
        i != j,
    ensures
        arch.entry_base(layer, base, i) == arch.entry_base(layer, base, j),
{
}
```
REASON: If two distinct indices map to the same entry base, page table entries would alias and the address space partitioning is broken
===PHI_END===
