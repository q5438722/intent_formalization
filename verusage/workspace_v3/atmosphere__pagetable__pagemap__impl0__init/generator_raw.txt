```verus
proof fn phi_1_init_has_present_entry(pm: PageMap)
    requires
        pm.wf(),
        forall|i: int| #![trigger pm@[i].is_empty()] 0 <= i < 512 ==> pm@[i].is_empty(),
    ensures
        exists|i: int| 0 <= i < 512 && pm@[i].perm.present,
{
}
```
REASON: After init all entries must be empty (present==false); if a present entry exists, init failed to clear the page map
