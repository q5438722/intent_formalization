        a.inv(),
        layer < a.layers.len(),

        a.upper_vaddr(layer, base) == base,

REASON: If upper_vaddr equals the base, the mapped address range has zero size, meaning the layer maps nothing—defeating the purpose of the page table level.

NAME: entry_base_ignores_base
TARGET_FN: entry_base
TYPE: logical
CODE:
```verus
proof fn phi_4_entry_base_ignores_base(a: Arch, layer: nat, base: nat)
    requires
        a.inv(),
        layer < a.layers.len(),
        base > 0,
    ensures
        a.entry_base(layer, base, 0) != base,
{
}
```
REASON: entry_base at index 0 must equal the base address; if it doesn't, the arithmetic is broken and all derived virtual addresses would be shifted incorrectly.
