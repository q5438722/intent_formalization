```verus
    requires
        x86_arch_spec.inv(),
        x86_arch_spec.layers.len() >= 4,
    ensures
        x86_arch_spec.entry_size(0) == x86_arch_spec.entry_size(3),
{
}
```
REASON: Layer 0 maps 512 GiB regions while layer 3 maps 4 KiB pages; equal entry sizes would collapse the page-table hierarchy into a single flat level.
