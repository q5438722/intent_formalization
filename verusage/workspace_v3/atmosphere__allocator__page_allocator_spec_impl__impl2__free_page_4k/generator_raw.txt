NAME: freed_page_becomes_mapped
TARGET_FN: free_page_4k
TYPE: behavioral
CODE:
```verus
proof fn phi_1_freed_page_becomes_mapped(old_alloc: PageAllocator, new_alloc: PageAllocator, target_ptr: PagePtr)
    requires
        old_alloc.wf(),
        old_alloc.allocated_pages_4k().contains(target_ptr),
        new_alloc.wf(),
        new_alloc.free_pages_4k() =~= old_alloc.free_pages_4k().insert(target_ptr),
        new_alloc.allocated_pages_4k() =~= old_alloc.allocated_pages_4k().remove(target_ptr),
        new_alloc.mapped_pages_4k() =~= old_alloc.mapped_pages_4k(),
        new_alloc.mapped_pages_2m() =~= old_alloc.mapped_pages_2m(),
        new_alloc.mapped_pages_1g() =~= old_alloc.mapped_pages_1g(),
    ensures
        new_alloc.page_is_mapped(target_ptr),
{
}
```
REASON: A freshly freed page appearing in mapped sets would violate state disjointness between free and mapped pages

```verus
        new_alloc.allocated_pages_4k() =~= old_alloc.allocated_pages_4k().remove(target_ptr),
    ensures
        !new_alloc.allocated_pages_4k().contains(other_ptr),
{
}
```
REASON: Freeing one page should never cause a different allocated page to disappear from the allocated set
