NAME: adjacent_entry_overlap
TARGET_FN: Arch::entry_base, Arch::next_entry_base
TYPE: boundary
CODE:
```verus
proof fn phi_1_adjacent_entry_overlap(arch: Arch, layer: nat, base: nat, i: nat)
    requires
        arch.inv(),
        layer < arch.layers.len(),
        i + 1 < arch.num_entries(layer),
    ensures
        arch.entry_base(layer, base, i + 1) < arch.next_entry_base(layer, base, i),
{
}
```
REASON: Adjacent entries overlapping would mean two page table entries map the same virtual address, destroying isolation.
