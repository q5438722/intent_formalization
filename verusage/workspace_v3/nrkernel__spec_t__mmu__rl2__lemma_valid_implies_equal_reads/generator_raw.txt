===PHI_START===
NAME: off_by_one_aligned
TARGET_FN: aligned
TYPE: behavioral
CODE:
```verus
proof fn phi_4_off_by_one_aligned(addr: nat, size: nat)
    requires
        aligned(addr, size),
        size > 1,
    ensures
        aligned((addr + 1) as nat, size),
{
}
```
REASON: Shifting an aligned address by 1 must break alignment for any size > 1; provability implies the spec is vacuous
===PHI_END===
