TYPE: logical
CODE:
```verus
proof fn phi_2_entry_base_collision_different_indices(a: Arch, layer: nat, base: nat, i: nat, j: nat)
    requires
        a.inv(),
        layer < a.layers.len(),
        i < a.num_entries(layer),
        j < a.num_entries(layer),
        i != j,
    ensures
        a.entry_base(layer, base, i) == a.entry_base(layer, base, j),
{
}
```
REASON: If two distinct entry indices in the same layer produce the same base address, page table entries would map overlapping virtual address regions, breaking isolation.
