NAME: aligned_symmetry
TARGET_FN: aligned
TYPE: logical
CODE:
```verus
proof fn phi_2_aligned_symmetry(a: nat)
    requires
        a > 0,
        aligned(a, PAGE_SIZE as nat),
    ensures
        aligned(PAGE_SIZE as nat, a),
{
}
```
REASON: Alignment is not symmetric; PAGE_SIZE is not a multiple of every page-aligned address (e.g. 4096 % 8192 != 0)
===PHI_END===

NAME: nonzero_smaller_than_size
TARGET_FN: aligned
TYPE: boundary
CODE:
```verus
proof fn phi_3_nonzero_smaller_than_size(a: nat, b: nat)
    requires
        0 < a,
        a < b,
    ensures
        aligned(a, b),
{
}
```
REASON: A positive value strictly less than its alignment size cannot be divisible by it; would break all alignment-size invariants
===PHI_END===

===PHI_START===
NAME: mul_closure
TARGET_FN: aligned
TYPE: logical
CODE:
```verus
proof fn phi_4_mul_closure(a: nat, b: nat, c: nat)
    requires
        b > 0,
        c > 0,
        aligned(a, b),
        aligned(a, c),
    ensures
        aligned(a, b * c),
{
}
```
REASON: Alignment to two sizes does not imply alignment to their product; e.g. aligned(12,4) && aligned(12,6) but not aligned(12,24)
===PHI_END===
