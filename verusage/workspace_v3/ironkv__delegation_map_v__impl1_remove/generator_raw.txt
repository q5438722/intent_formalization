```verus
proof fn phi_1_length_unchanged_after_remove<K: KeyTrait + VerusClone>(old_self: StrictlyOrderedVec<K>, post_self: StrictlyOrderedVec<K>, k: K, i: usize)
    requires
        old_self.valid(),
        i < old_self@.len(),
        post_self.valid(),
        k == old_self@.index(i as int),
        post_self@ == old_self@.remove(i as int),
        post_self@.to_set() == old_self@.to_set().remove(k),
    ensures
        post_self@.len() == old_self@.len(),
{
}
```
REASON: Remove must decrease the length by exactly one; an unchanged length means the element was not actually removed.
