```verus
proof fn phi_1_always_none(pt: PageTable, target_l4i: L4Index, target_l3i: L3Index, l4_entry: PageEntry)
    requires
        pt.wf(),
        pt.kernel_l4_end <= target_l4i < 512,
        0 <= target_l3i < 512,
        pt.spec_resolve_mapping_l4(target_l4i) =~= Some(l4_entry),
    ensures
        pt.spec_resolve_mapping_l3(target_l4i, target_l3i).is_None(),
{
}
```
REASON: If L3 resolution always returns None, the function is vacuous and no L3/L2/L1 page walks can ever succeed
