TYPE: behavioral
CODE:
```verus
proof fn phi_1_entry_base_collision(arch: Arch, layer: nat, base: nat, i: nat, j: nat)
    requires
        arch.inv(),
        layer < arch.layers.len(),
        i < arch.num_entries(layer),
        j < arch.num_entries(layer),
        i != j,
    ensures
        arch.entry_base(layer, base, i) == arch.entry_base(layer, base, j),
{
}
```
REASON: Different indices should map to distinct base addresses; collision would allow overlapping memory regions
