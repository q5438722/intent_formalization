NAME: set_changes_seq_length
TARGET_FN: set
TYPE: behavioral
CODE:
```verus
proof fn phi_1_set_changes_seq_length<A, const N: usize>(pre: Array<A, N>, post: Array<A, N>, i: usize, out: A)
    requires
        0 <= i < N,
        pre.wf(),
        post.seq@ =~= pre.seq@.update(i as int, out),
        post.wf(),
    ensures
        post@.len() != pre@.len(),
{
}
```
REASON: Array::set should preserve sequence length; both pre and post satisfy wf so length must remain N
===PHI_END===

TARGET_FN: set
TYPE: boundary
CODE:
```verus
proof fn phi_2_set_corrupts_other_index<A, const N: usize>(pre: Array<A, N>, post: Array<A, N>, i: usize, j: usize, out: A)
    requires
        0 <= i < N,
        0 <= j < N,
        i != j,
        pre.wf(),
        post.seq@ =~= pre.seq@.update(i as int, out),
        post.wf(),
    ensures
        post@[j as int] != pre@[j as int],
{
}
```
REASON: Updating index i must not alter any other index j; Seq::update preserves elements at positions != i
===PHI_END===
