```verus
    requires
        ptr1 % 0x1000 == 0,
        ptr2 % 0x1000 == 0,

        spec_page_ptr2page_index(ptr1) == spec_page_ptr2page_index(ptr2),
{
}
```
REASON: If two distinct page-aligned pointers map to the same index, the allocator cannot distinguish pages and would corrupt state
